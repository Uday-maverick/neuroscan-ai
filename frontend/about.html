About This System

Model Information

Our AI-powered classification system uses a state-of-the-art ConvNeXt Base deep learning architecture trained on thousands of MRI brain scans. The model achieves exceptional accuracy in detecting and classifying different stages of Alzheimer's disease.

Key Features

  • Deep Learning Architecture: ConvNeXt Base
  • Training Dataset: 5,124 MRI scans
  • Validation Method: 5-fold cross-validation
  • Data Augmentation: Horizontal flip, rotation, brightness/contrast adjustment
  • Ensemble Approach: TTA + Ensemble

Technology Stack

  • Deep Learning: PyTorch 2.1+ with ConvNeXt architecture
  • Backend: Flask 3.0 web framework
  • Image Processing: Albumentations, OpenCV, Pillow
  • Explainability: Grad-CAM++ visualizations
  • Visualization: Matplotlib, Seaborn, Plotly
  • Model Library: timm (PyTorch Image Models)
